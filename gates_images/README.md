# Папка для изображений ворот и парковки

В этой папке хранятся изображения ворот и парковочных мест.

## Структура

- `АБК-1/` - изображения ворот АБК-1 (номера 1-59, 66-83)
- `АБК-2/` - изображения ворот АБК-2 (номера 1-10)
- `Парковка/` - изображения парковочных мест (номера 1-50)

## Формат именования файлов

Файлы должны быть названы по номеру ворот/места:
- `1.jpg`, `2.png`, `3.jpeg` и т.д.
- Можно использовать ведущий ноль: `01.jpg`, `02.png`

## Поддерживаемые форматы

- `.jpg`, `.jpeg`
- `.png`
- `.gif`
- `.bmp`
- `.webp`

## ⚠️ Важно

Изображения могут содержать конфиденциальную информацию.
При необходимости загружайте их непосредственно на сервер,
минуя публичный репозиторий.
